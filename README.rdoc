= Lolita Menu
Create, sort and publish menus with Lolita.

==Instalation
In your project Gemfile put
  gem 'lolita-menu'
And run
  bundle install
Like with other Lolita gems call
  rails g lolita_menu:install
Since this gem comes with migrations
  rake db:migrate

Enjoy!

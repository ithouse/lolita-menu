= lolita-menu

== Instalation
In your Gemfile put 
    gem 'lolita-menu'
And run
    bundle install
After that, like with other Lolita gems, call.
    rails g lolita_menu:install
Since this gem comes with migration you should run
    rake db:migrate
Enjoy!
